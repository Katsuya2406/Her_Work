SHARED MISSION
Websites and web technologies have become an important aspect in the life of modern society. People now rely on web technologies for different aspects such as health, education, entertainment, interactions with other people, and many more. Websites such as YouTube could be used for entertainment, while websites such as Amazon could be used for commerce. Some websites could also be used as a way of income. Websites such as Shopee, Lazada, eBay, and others, allow users sell items and products. Technology has integrated itself into society that people rely on websites for many things. Basic needs such as food, clothing, and shelter can be found online. Website structures are used as a word format layout of what a website should contain and how it should flow. Its structure includes the homepage, webpages, about page, contact page, webpage containing the main content of the website, and more. The example shown in figure 2 is known as the hierarchical structure or tree model. According to Figma it is the most common structure as it provides a clear and easy to understand model. It is best used for E-commers, business and informational websites. The model shown in Figure 3 is a linear or sequential model that presents each page in a fixed order. It guides the users through the content using a pre-defined path. It moves the user to the next page/step using a next and previous navigation. This model is best used for content such as tutorials, flows, or multi-step forms. This model is mostly used for educational sites, online courses, walkthroughs, sign-up pages, and more. The model shown in Figure 4 is known as a Database/Dynamic model. It relies on a database to generate pages based on user queries and/or behavior. This model focuses less on structure and focuses or organizing content based on filters, searches, and user interaction. This model is common for large websites with large amounts of data. This model is best used for E-commerce websites with numerous products, news sites, large Content Management System (CMS) platforms, or any websites requiring real-time content based on user input. According to TechTerms “The client-server model describes how a server provides resources and services to one or more clients. Examples of servers include web servers, mail servers, and file servers. Each of these servers provide resources to client devices, such as desktop computers, laptops, tablets, and smartphones. Most servers have a one-to-many relationship with clients, meaning a single server can provide resources to multiple clients at one time.”   Front-End is the part of a website that the users sees and interacts with directly. It relies of HTML, CSS, Javascript, Frameworks: React, Angular, and Vue.js. Its main role is to create a layout and design the User Interface (UI). Back-End is the server-side logic that runs and powers the website. It relies on technologies such as Python. PHP, Java, Node.js. Databases such as MySQL and MongoDB. Its main role is to process and store the data from the users, authentication and authorization, and server configuration. Websites uses protocols such as HTTP and HTTPS to transfer data between the browsers and the servers. It standardizes rules governing how data is transmitted, formatted, processed, stored, and received across the internet.
